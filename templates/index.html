What we've done in Y2 CS so far

We're almost halfway through MEET Y2 CS! You will be starting work on the projects soon.

We started off learning about HTML and CSS so that we could make websites. This is called the "frontend", and it's the part of the webapp that your users see when they log on.

Then, we did learned about relational databases. We connected Python with databases using something called a package call SQLAlchemy. Databases store all the information for your webapp.

After talking about databases, we learned how to build a webserver using Flask. Web servers allow you to create dynamic websites! Each urls of a website trigger a specific function in the backend. The function can retrieve data from the database or just run python code. Then Jinja, the template rendering part of Flask, allows us to render template and pass variable to HTML.

Now, we're learning JavaScript. We can make dynamic pages that change when we click on things or type data in boxes.

Technologies that you know:

  • HTML
  • CSS
  • Python
  • SQLAlchemy
  • Flask
  • JavaScript
  • jQuery
